## Fraunces

- Fichiers : `Fraunces.ttf`, `Fraunces-Italic.ttf`
- Type : variable fonts (axes `SOFT`, `WONK`, `opsz`, `wght`)
- Auteur : Undercase Type (Phaedra Charles, Flavia Zimbardi)
- Source : https://github.com/googlefonts/fraunces
- Usage projet : titres et textes éditoriaux (refonte UI Cottagecore 2026-04)

## IBM Plex Mono

- Fichiers : `IBMPlexMono-Regular.ttf`, `IBMPlexMono-SemiBold.ttf`
- Auteur : Mike Abbink, Paul van der Laan, Pieter van Rosmalen (IBM / Bold Monday)
- Source : https://github.com/IBM/plex
- Usage projet : HUD, métriques, données techniques
